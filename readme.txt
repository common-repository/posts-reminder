== Description ==

Posts reminder notifies you when you forget to write a post on the current day. It notifies you at 22:00 to the email you
specifiy on the options page. 

== Installation ==

1. Unzip the content of the archive into the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Personalize the plugin into the posts reminder options page (inside options)
